# Pagination

## Purpose

Pagination moves through paged datasets such as designs, templates, orders, publications, or admin lists.

## Anatomy

- Nav root with `aria-label="Pagination"`.
- List and item wrappers.
- Page links.
- Previous/next controls.
- Ellipsis helper.

## Accessibility

Active page uses `aria-current="page"`. Previous and next controls include explicit accessible labels.
