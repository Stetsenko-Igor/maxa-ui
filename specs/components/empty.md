# Empty

## Purpose

Empty communicates that a view has no results or no content yet, then offers the next useful action.

## Anatomy

- Optional decorative icon.
- Title.
- Optional description.
- Optional primary and secondary actions.

## Variants

- Size: `sm`, `md`, `lg`.

## Product Notes

Maxa already uses empty states such as “Scheduled Post Not Found”. This component standardizes that pattern for search results, dashboard folders, carts, and generated content flows.
