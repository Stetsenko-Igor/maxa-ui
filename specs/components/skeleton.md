# Skeleton

## Purpose

Skeleton reserves layout while content is loading. Use it for cards, list rows, thumbnails, text lines, and dashboard placeholders.

## Anatomy

- Root: decorative placeholder block.
- Shimmer: token-driven highlight animation.

## Variants

- `text`: single-line placeholder.
- `rect`: card, thumbnail, input, or block placeholder.
- `circle`: avatar or icon placeholder.

## Accessibility

Skeleton is decorative and always `aria-hidden`. Pair with a parent region or Spinner when users need an announced loading state.
