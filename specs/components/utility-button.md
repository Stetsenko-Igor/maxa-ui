# Utility Button

## Purpose

Utility Button is a compact icon-only square action for dense surfaces such as toolbars and view controls.

## Anatomy

- Icon-only button surface.
- Required accessible label.

## Variants

- Size: `sm`, `md`, `lg`.
- State: default, selected, disabled.

## Accessibility

Every Utility Button must provide `aria-label`. Use `selected` only for persistent toggle-like state.
