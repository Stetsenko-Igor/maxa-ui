# Tabs

## Purpose

Tabs switch between related panels without leaving the current page. Use them for compact editor panels, settings sections, package views, and documentation examples.

## Anatomy

- Root: manages selected value.
- List: tab trigger container.
- Trigger: selectable tab.
- Content: associated panel.

## Variants

- Horizontal button tabs only.
- Active tab uses dark fill; inactive tabs use neutral gray fill.
- Active, hover, focus, and disabled states.

## Accessibility

Built on Radix Tabs for keyboard navigation and ARIA relationships.
